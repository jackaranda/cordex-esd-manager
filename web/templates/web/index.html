CORDEX Experiment Manager

Welcome to the CORDEX Experiment Manager web platform. This platform facilitates the design, configuration, and management of coordinated numerical experiments. Currently the platform is focussed on the CORDEX Empirical Statistical (CORDEX-ESD) project but it is intended that the platform could be extended to support a wide range of coordinated numerical experiments.

Using the platform

The manager consists of two main sections, experiments, and submissions. These are always accesible through the menu at the top of the page.

Experiments

The first is experiments. This section lists the currently open experiments and allows details of each experiment to be viewed. Admistrators are able to create new experiments or edit existing experiments. Participants are only able to view experiments and add comments/questions to experiments.

Submissions

The second section is the submission section. This section allows a partcipant to submit results to a particular experiment. Results submissions take place through uploads such that a single submission may consist of multiple uploads. Each upload contributes to the final submission for that participant for an experiment. Each time an upload is added to a submission the submission is re-processed to check it for compliance with the experiment configuration, convert it to a standard data format, and merge dates and variables as needed. The processing takes place in the background and current status of all a partcipants submissions can be viewed at any time in the submission section of the platform.
